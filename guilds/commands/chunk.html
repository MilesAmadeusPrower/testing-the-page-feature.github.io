/chunk perm

This is your goto tool in order to observe all of the permissions contained in the chunk you are standing on.

/chunk edit

This is a GUI command editor to be able to change all of the related permissions in the chunk, granted if you have access to it.

/chunk fortify [#]

Every claimed chunk is considered a fort. But by default, an unfortified chunk has next to no capture HP and is therefore not able to fend off anybody. This command is used when you want to invest cobblestone into higher capture HP to be harder to take over during a war.

/chunk repair

In event a chunk is either taken over, or sieged, you are able to use this command to rebuild.
